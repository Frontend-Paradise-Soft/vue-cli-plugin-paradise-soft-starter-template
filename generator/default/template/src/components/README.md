# Components

管理組件的文件夾

## 資料夾結構範例

```
├── /components/
│   ├── /common/                # 未分類組件
│       └── /AvatarImage.vue
│   ├── /ui/                    # UI 組件
│       ├── /BaseButton.vue
│       └── /BaseInput.vue
│   ├── /layout/                # 關於 layout 的組件
│       ├── /DefaultLayoutHeader.vue
│       └── /DefaultLayoutFooter.vue
│   └── /pages/                 # 關於 pages 的組件
│       └── /article/               
│           ├── /ArticleList.vue
│           └── /ArticleListItem.vue
├── ...
```
